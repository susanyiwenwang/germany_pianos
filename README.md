# An Analysis on Piano Production
### Data Analytics Project on the Piano Market in Germany

## Key Question:
- How has the digital keyboard and international trade affected the piano manufacturing industry in Germany?
  
## Data Sources
- World Integrated Trade Solutions
- Destatis GENESIS Online
  
## Analysis
- Exploratory Analysis on instrument distribution
- Trends over 10 year period

## Project Deliverables
Findings have been summarized in an infographic to be easily distributed on social media such as LinkedIn and Facebook.
